第一章   蹊跷<br>
    三月,又是三月,周毅坐在原本荒芜的山坡上,而这儿如今已是一片绿色,绿色...蔓延到很远.			<br>
    这儿本是个不大不小的村子,偏巧最近又发生了一件不大不小的事情.<br>
    村子里的人个个习武,虽然很少在江湖上走动,但这历代沿袭下来的传统,也使得村子在江湖上有了不大不小的名气.<br>
    每个村子都会有那么一两家人很有名,就像每个镖局都有个镖头.没了镖头固然保不了镖,但没了那有名的一两家人,村子的人便会无所事事,终日你争我斗.毕竟大家是靠体力吃饭的,但哪家都不会因为家里人练过功夫,而能每天比别人多耕几十亩地,所以见了好处,难免人人眼红.<br>
    这个村子当然也不例外,这个村子也有那么两户很有名的人家,村南头的周家和村北头的吴家.<br>
    周家是武术世家,上可追溯到清代重阳七子之首王重阳的师弟周伯通,下可到昔日以一手开碑掌剿灭方圆十里土匪的周开泰,周家在村子人心目中,无疑是本村一块响当当的招牌.吴家的人自然也习武,但却祖上几辈都与朝廷打过交道，甚至当过大官，是以村里人对吴家更多的是敬畏，却少了些许景仰.<br>
    俗话说,一山容不得二虎.一个村子自也容不下两个显赫的家族.村里人也明白这一点,所以也有了条不成文的规矩,无论谁都不会在周家人面前主动向吴家的人打招呼,那无疑是不把吴家放在眼里,却更没有人有胆在两家人面前互相问好的,那样,分明是给了两家人莫大的难看,而这造成的结果便是,今后的日子一定别想好好过.<br>
    两家就这样明争暗斗着,村里人心里也雪亮,却没人知道这暗下的勾心斗角什么时候开的头,也没人说的清哪个时候是个了结.只知道周家祖上划了几百亩地建了骨头山庄,吴家便立马开山凿石筑了万丈高的吴家堡.周家的大少娶媳妇的时候骨头山庄上下一片欢腾,据说庄主特地为儿子准备了几百桌上好的酒席,光是打发出去请客的下人就走了好几千,请的也都是江湖上赫赫有名的一代宗师,更有神刀门的掌门"魔刀"丁彭亲自登门贺喜,这着实给了周家的人莫大的面子,谁都知道,丁彭的刀法当真已趋于化境,没人知道江湖上谁能接得住他魔刀一劈,因为谁都不想被人活生生从中间分成两瓣.吴家也不示弱,就在周家大少洞房的第二天,吴家举家游玩西湖,据当时在场的老人回忆,村里唯一的一条大路被吴家雇的马车给硬生生塞满了.马是上好的西陆良驹,车一律漆成黑色,车夫也是一色的玄色劲装,在阳光下散发着狂野的气息.无论是谁,接了这份差事都会变的训练有素的,毕竟,每辆车的赏钱,是十两黄金,老人说到这里,也是浑身一个激灵.<br>
    按说这两家该是没有什么烦心的事了,至少在村里,他们都该活的比谁都快活.<br>
    周毅是周家的二少,自幼便很受长辈们器重,说他是块练武的好料子.周毅便是在这种关爱与呵护下长大的,哪怕曾经有无数的家奴好心劝告他江湖有多么险恶,更是有数不清的长辈告戒他将来有一日行走江湖时要多留几个心眼,他也曾经天真的认为,天下的人都会对他像府上一样必恭必敬.<br>
    可那终究是曾经,曾经的意思就是,就在昨天,周家一家人在一夜之间竟然都消失的干干净净.庄上没有打斗过的痕迹,金库里的纹银没有少了分毫.可人却不见了,都不见了,连府上的家仆都不例外.<br>
    周毅并不喜欢仆人,所以等到他发现庄上是实实在在没有人时,已是正午.<br>
    这时周毅坐在草地上,双眼盯着空洞的天空发愣.原本他是很喜欢每天这个时候,在这个地方晒太阳的,可今天却是个例外,大大的例外.现在他的心里很乱,乱到甚至想到一剑抹断自己的脖子.<br>
    "爹,娘,你们在哪里?"周毅用力把手上的碎草丢到空中,被风吹到很远.<br>

第二章   冤家<br>

    "好久不见,你的武功好象精进不少."声音是从身后传来.对于能在发完暗器后还在自己眼皮子底下转到自己身后的人,周毅本该大吃一惊才是,可是他现在却在笑,而且笑的很好看.<br>
    任谁在经历了刚才那可怕的一幕,都不会笑的这么轻松,可周毅不同,他很随便,随便到可以玩笑自己的生死.<br>

    可他马上后悔了,他忘了答案是只用听的,转过脸来的结果只有一个,就是面对离自己不到一寸的一双闪亮的大眼睛.<br>
